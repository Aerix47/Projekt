# Nettside spill projekt
Jeg skal lage et spill som er del av en webside. Spillet skal være en cookie-clicker clone som handler om å clicke en JavaScript logo. Det kommer til å være forskjelige oppgraderinger (flavortext er IT relatert) og metoder som hjelper deg til å progresse gjennom spillet. Programmering språkene kommer til å bli HTML, CSS for siden og hoved språket for spillet skal være JavaScript. Jeg skal bruke flask og MySQL til å lagre data og poeng. Raspberry PI skal bli brukt som en database. Jeg vuderte Dbeaver for å lagre data fordi det så interesant ut, men jeg valgt til å bruke MySQL istenden.

Jeg kan vise kompentanse gjennom programmering, dokumentering, frontend, backend, database, flask, datalagring, UI, UEX.
